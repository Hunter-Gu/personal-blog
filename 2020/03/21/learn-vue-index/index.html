Nail's Blog

vue

最近学习了一下 Vue 的源码，花了不少时间，记录在了 github pages。

由于是第一次看框架类的源码，下面是对看源码过程的总结。

前提

看源码是有前提的，你需要对框架足够熟悉了再去看，大致可以分为如下几个问题。

Vue 解决了什么问题?

  • 组件化
  • 声明式渲染
  • MVVM

Vue 的设计思想是什么？

  • 响应式数据

Vue 有哪些主要的功能点？

  • 变化监测
  • 模板编译
  • 虚拟 DOM
